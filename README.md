# 🎓 Student Management System - Frontend

## 📌 Overview
This is a simple and clean **Student Management System frontend** built using **HTML, CSS, and JavaScript**.  
It allows users to manage student records with full CRUD operations using a REST API backend.

---

## 🖼️ UI Preview

![Student Management UI](./screenshot.png)

---

## 🚀 Features
- ➕ Add new students
- 📋 View all students
- ✏️ Edit student details
- ❌ Delete students
- 🔍 Search students
- 🎨 Clean and modern UI

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Fetch API
